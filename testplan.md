# Sound Scheduler Beta Test Plan

## 1. Installation
- Confirm app installs cleanly on target devices.
- Confirm first launch shows correct UI and permissions.

## 2. Free Version Tests
- Can create up to 3 routines.
- Ads appear on create/delete actions.
- Cannot create a 4th routine without upgrade.
- Check ads are minimally invasive and do not block usage.

## 3. Premium Upgrade
- Purchase via Google Play Billing.
- Ads are disabled immediately.
- Routine limit increases to 10.
- Access to:
  - Dark/AMOLED theme.
  - Widget support.
  - Offline backup and restore.
  - Archive completed routines.
  - Routine templates.

## 4. Location-Based Routines
- Create location-based routine.
- Test geofence triggers notification on entry/exit.

## 5. Calendar-Based Routines
- Create calendar-based routine.
- Test notification near event start.

## 6. Notifications
- Test time-based, location-based, and calendar-based notifications.

## 7. Backup and Restore
- Create multiple routines.
- Backup data.
- Clear app data, restore data.
- Confirm routines are restored correctly.

## 8. Widget
- Add widget to home screen.
- Confirm display and tap functionality.

## 9. Archive Feature
- Archive a completed routine.
- Retrieve archived routines.

## 10. Stability
- Long run app without crashes.
- Test screen rotations, background, and foreground transitions.

## 11. Power Management
- Ensure geofencing and calendar checks do not drain battery.
- Confirm notifications still trigger reliably.

---

**Expected Outcome:** Pass all tests before Play Store beta publishing.
